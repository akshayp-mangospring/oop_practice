### To run programs in Java
```
cd java
javac ./*.java
java Main
```
### To run programs in Ruby
```
cd ruby
ruby ./main.rb
```
### To run programs in Python 2
```
cd python
python ./main.py
```
### To run programs in Python 3
```
cd python
python3 ./main.py
```
### To run programs in Node JS
```
cd node
node ./stack.js
```
### To run programs in Go
```
cd go
go run ./stack.go
```
